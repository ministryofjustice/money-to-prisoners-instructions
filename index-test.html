Send money to someone in prison

TEST This is a test site

You can use this service to make a payment by debit card. We accept Visa, Mastercard and Maestro debit cards.

You can also find out how to make a UK bank transfer to a prisoner by:

  • online banking
  • mobile banking app
  • telephone banking

Your money won’t reach the prisoner if you make a transfer from a bank branch or non-UK account.

This service is free and secure, and available in all public prisons in England and Wales. You can’t use the service to send money to someone in a private prison.

Start now

Before you start

You’ll need the:

  • prisoner’s date of birth
  • prisoner number

Money usually takes around 3 working days to reach a prisoner’s account.

You need a bank account to use this service. If you don’t have one you can send money to a prisoner using a postal order, a cheque or cash.

Help using the service

You can get help if you’ve already sent money but it hasn’t reached the prisoner yet.

You can also contact the support team if you have any other questions about using the service.
