Send money to a prisoner

You can use your bank account to send money to a prisoner. You'll need:

  • a payment reference to identify the prisoner
  • the prison service bank account and sort code

You can only use this service if you’re sending money to prisoners in HMP Brixton, C Wing.

Payment reference

Use the:

  • Prisoner number
  • Prisoner date of birth

If you don’t know the prisoner number, the prisoner will be able to tell you what it is.

An example of how to write the payment reference

  • If the prisoner number is:

    A1234JG

  • And their date of birth is:

    10 January 1959

  • You'd write:

    A1234JG 10/01/1959

Bank details

All public prisons in England and Wales use the same bank account to receive money sent to prisoners. Make a payment using these details:

The prison service account number
48301337
The prison service sort code
45-50-55

The money should arrive in the prisoner's account in 2 working days*.

If you have any questions about how to make a payment you should contact your bank or building society.

*Your bank may take longer to process payments (up to 3 working days).
